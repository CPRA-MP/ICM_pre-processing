# ICM Pre-processing Scripts: Flow Calculators

This Flow Calculators repository contains several Python scripts that are used to compile observed USGS flows into a composite file that is then used to build daily hydrographs for future scenarios. These future scenario hydrographs are produced from a monthly wetness index generated from CMIP6 precipitation projections under various SSP/RCP combinations. 

```
/<api_version>/datasets/<variable>/<geographic_unit>/<time_unit>/data.<format>
```
